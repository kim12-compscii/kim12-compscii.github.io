What is a div element

What is one way you can identify or "target" a div element

A div can be identified or “targeted” in several different ways. For today, we are going to focus on classes. When you are calling a class in CSS, it will target every div with the same class. You can repeat styling across a website using the same class without having to repeat the CSS.


Why would you use a div element

To target a specific div,call the class. All div elements that share a classwill have the same styling.The class is case sensitiveand there must be a period before it. div if you would like to target specific elements in a div, call it in the CSS.


What is a border property

The border property allows you to style the border around an element. The border indicates the size of an element. The border-style property can have up to four values for each of the four sides of the border. The order is: Top, Right, Bottom, Left

What is the difference between margins and padding

A margin is the spacing around an element from everything around it. The margin begins at the element's border.Padding is the spacing content has from the element border.
